## 🔴 必須（Google広告運用の前に対応）

### 1. GA4 でコンバージョンイベントをマーク

- **現状**: `generate_lead` イベントはコードで発火するが、GA4 管理画面でコンバージョンとしてマークが必要
- **対応手順**:
  1. GA4 → **管理** → **イベント**
  2. `generate_lead` の右側の **「コンバージョンとしてマーク」** をON
- **所要時間**: 1分

### 2. Google Ads と GA4 のリンク

- **現状**: 未リンク
- **重要性**: 広告のCPA（顧客獲得単価）を計測するために必須
- **対応手順**:
  1. GA4 → **管理** → **Google Ads のリンク** → リンクを作成
  2. Google Ads 側 → **ツールと設定** → **コンバージョン** → **インポート** → GA4 の `generate_lead` を選択
- **前提**: Google Ads アカウントの作成が必要

## 🟡 推奨（広告運用開始後に順次対応）

### 3. ページ表示速度の最適化

- **現状**: CSS(69KB) / JS(11KB) が未圧縮
- **広告への影響**: ページ速度が遅いと広告の品質スコアが下がり、CPC（クリック単価）が上がる
- **対応候補**:
  - CSS/JS の minify
  - Critical CSS のインライン化
  - 不要な CSS ルールの削除

### 4. サンクスページの作成

- **現状**: フォーム送信後はインラインの完了メッセージのみ
- **広告への影響**: サンクスページがあると Google Ads のコンバージョン計測がより正確になる
- **対応**: 専用の `thanks.html` を作成 → GA4/Google Ads でサンクスページ到達をCV測定
